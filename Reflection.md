### 1. How did you approach creating more flexible functions with parameters?

- I tried to avoid hardcoding values and instead pass information into my functions so they could be reused for different situations. Using parameters made the functions more flexible because the same logic could work with different names or numbers. I also added simple checks to make sure the inputs made sense before doing anything else.
